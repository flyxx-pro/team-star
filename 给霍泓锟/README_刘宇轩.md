## 三、你需要重点改进的

### 问题1：认缴和转让混淆（最严重）
云汉35条(刘15)、三协17条(刘3) → 你的AI把大量转让/ESOP误分为认缴。
**建议**: 在 raw_response 分类后，加一层规则过滤：关键词含"转让/受让/代持解除" → 不进入认缴。

### 问题2：赛分 rowspan 展开后列匹配偏移
赛分10条(刘15条) → rowspan展开后列位置可能偏移。
**建议**: 展开后打印每行的列数验证，确保都是6列。

### 问题3：段落文字的增资
聚贝投资只在段落中，不在表格里 → 你的AI可能漏掉了。
**建议**: AI提取时不仅读表格，也搜索"增发股份"+"新增股东"等段落模式。

### 问题4：Cross-check 格式
你的 cross_check.csv 是简化版，老师要求13列标准格式。参考友升示范 Excel 的 sheet 3。
