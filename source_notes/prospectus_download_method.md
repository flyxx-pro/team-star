# 招股说明书下载方法

## 数据源

巨潮资讯网 (cninfo.com.cn) — 中国证监会指定信息披露网站

## API 接口

- **端点**: POST http://www.cninfo.com.cn/new/hisAnnouncement/query
- **关键参数**: searchkey="{公司简称} 招股说明书"（不加 column 参数）
- **日期范围**: seDate=2024-01-01~2025-12-31

## 反爬策略

1. Session Cookie：先 GET 首页获取 Cookie
2. User-Agent 轮换：4个UA随机切换
3. 随机延时：API请求间隔 0.5-1.5s，PDF下载间隔 2-5s
4. 重试机制：最多3次，每次重试间隔 3-8s
5. 完整性校验：下载后检查文件≥100KB，否则删除重试
6. 断点续爬：检查本地文件是否已存在且完整

## 文件命名

`{股票代码}_{公司简称}_招股说明书.pdf`

## 版本选择

正式稿 > 注册稿 > 申报稿 > 上会稿

## MinerU 解析

- 版本: v3.1.11
- 页码范围: 每家公司不超过200页
- 策略: 1-50页(目录+基本信息) + VC章节区域
