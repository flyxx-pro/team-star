# 有限责任公司专用 Prompt 设计

> 解决黄山谷捷、三联锻造等有限责任公司出资记录格式与股份公司不一致的问题

---

## 一、问题根因

| 披露格式 | 有限责任公司 | 股份有限公司 |
|---------|-----------|-----------|
| 出资记录 | "孙国奉出资175万元，占注册资本35%" | "策星银河认购279万股，认购金额1757.7万元" |
| 单位 | 万元注册资本 | 万股 |
| 增资表述 | "增加注册资本至XX万元" | "增发XX万股" |
| 价格 | "每元注册资本XX元" | "每股XX元" |
| 股权转让 | "将其持有的XX万元出资额转让" | "将其持有的XX万股股份转让" |

如果不区分公司类型，AI会：
- 黄山谷捷：把"出资额"误填为"认购数量(万股)"
- 三联锻造：找不到"认购数量"和"认购金额"，全部填null
- 三协电机（有限公司阶段）：股权转让事件无法识别

---

## 二、通用Prompt（自动判断公司类型）

```
你是专业的招股书数据分析师。请从文本中提取认缴流量和股权结构存量。

## 公司类型判断
首先判断目标公司当前是有限责任公司还是股份有限公司：
- 有限责任公司：出资记录使用"出资额（万元注册资本）"表述，注册资本为XX万元
- 股份有限公司：出资记录使用"持股数（万股）"表述，股本为XX万股
- 如果公司在历史上从有限责任公司变更为股份有限公司（整体变更/股改），早期事件按有限责任公司规则处理，股改后事件按股份有限公司规则处理

## 有限责任公司规则
字段映射：
1. "出资额（万元注册资本）" = "认购金额（万元）"（两者在数值上相等）
2. "出资比例（%）" = "持股比例（%）"（两者含义相同）
3. "认购数量（万股）"字段留空（有限责任公司没有"股"的概念）
4. 如果原文写"每元注册资本XX元"，填入"认购价格（元/股）"字段
5. 股权转让事件中，"转让出资额（万元）"填入"转让数量（万股）"字段

示例：
原文："孙国奉出资175万元，占注册资本35%"
提取：
- 认购方：孙国奉
- 认购数量（万股）：null（有限责任公司阶段）
- 认购金额（万元）：175
- 持股比例（%）：35

## 股份有限公司规则
按标准格式提取即可。

## 整体变更（股改）处理
股改不是增资事件，不进入认缴流量表。但股改后的股权结构应作为存量快照。
如果原文写"以净资产折股的方式缴纳注册资本XX万元"：
- record_type: equity_snapshot
- 时点: 股改后
- 总股本(万股): XX（股改后的新注册资本）
- 各股东的持股数和比例按股改后的数据填写
```

## 三、有限责任公司专用System Prompt（精简版，直接喂AI）

```
你正在处理一家有限责任公司的招股书。有限责任公司的出资记录与股份公司不同：

1. 没有"认购数量（万股）"概念 → 此字段始终填null
2. "出资额（万元注册资本）"即等于"认购金额（万元）"
3. "出资比例"即等于"持股比例"
4. 股权转让以"转让出资额"表述 → 填入"转让价款（万元）"
5. 增资以"增加注册资本至XX万元"表述 → 保持原文本表述
6. 如果公司后期整体变更为股份有限公司，股改时点之后的记录按股份公司规则处理

记住：有限责任公司的"出资额"和"认购金额"是同一个数字，不要留null。
```

## 四、股份有限公司专用System Prompt

```
你正在处理一家股份有限公司的招股书。按标准格式提取：

1. 认购数量：按"万股"填写，原文是"股"的除以10000
2. 认购金额：按"万元"填写，原文是"元"的除以10000
3. 每股价格：填入"元/股"
4. 股权转让按标准股份转让格式提取
```

## 五、实际应用对比

### 三联锻造（有限责任公司→股份公司）

**股改前事件（2004年设立）**：
- 用有限责任公司Prompt → 出资额175万=认购金额175万，认购数量=null
- 结果：孙国奉 | 认购数量null | 认购金额175万 | 比例35%

**股改后事件（2019年增资）**：
- 用股份有限公司Prompt → 三联合伙认购348万股，每股4.50元
- 结果：三联合伙 | 认购数量348万 | 认购金额1566万 | 价格4.50

### 黄山谷捷（有限责任公司）

**增资事件**：
- 赛格高技术"认缴462.8571万元" → 认购金额=462.8571万，认购数量=null
- 上汽科技"认缴51.4286万元" → 认购金额=51.4286万，认购数量=null
- 黄山佳捷"认缴90.2256万元" → 认购金额=90.2256万，认购数量=null

**股权转让**：
- "黄山供销集团将780万元出资额转让" → 转让价款=780万

## 六、自动化判断逻辑

在调用AI前，先用规则扫描文本判断公司类型：

```python
def detect_company_type(text):
    llc_signals = ['出资额', '注册资本', '每元注册资本', '万元注册资本']
    jsc_signals = ['持股数', '万股', '每股', '股份数量', '股']

    llc_score = sum(1 for s in llc_signals if s in text)
    jsc_score = sum(1 for s in jsc_signals if s in text)

    if '整体变更' in text or '股改' in text:
        return 'mixed'  # 有过股改，分阶段处理

    return 'llc' if llc_score > jsc_score else 'jsc'
```

如果判断为 `mixed`，则按"股改"时点切分文本，前半段用有限责任公司Prompt，后半段用股份有限公司Prompt。
