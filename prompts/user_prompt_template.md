# User Prompt 模板

```
公司股票代码：{code}
公司简称：{company}
上市板块：{board}

以下是招股说明书章节文本（含页码标记）：

---
{extracted_text}
---

请提取认缴流量和股权结构存量数据，以JSON数组返回。
```

## 变量说明

- `{code}`: 6位股票代码，如 603418
- `{company}`: 公司简称，如 友升股份
- `{board}`: 板块，如 主板/创业板/科创板/北交所
- `{extracted_text}`: 关键词密度评分后的候选文本（含[第X页]标记），约35K-50K字符

## 分轮调用版本（数据量大时使用）

```
# 第一轮：只提取认缴流量
...请只提取认缴流量(subscription_flow)数据。每个认购方一行。只返回JSON数组。

# 第二轮：只提取股权存量
...请只提取股权结构存量(equity_snapshot)数据。每个时点每个股东一行。只返回JSON数组。
```
